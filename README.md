# PageKite API

This is a Node.js wrapper for PageKite [XML-RPC API](http://pagekite.net/support/service_api_reference/). Supported API are listed below

## Installation
```
npm install pagekite-api
```

## Basic Usage

```javascript
var Pagekite = require('pagekite-api');
```
